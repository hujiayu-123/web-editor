# 🎨 网页随心改 - 书签小工具

一个无需安装的浏览器书签工具,让你可以在任何网页上随意编辑内容、修改样式,就像使用Word一样简单!

## ✨ 特点

- 🚀 **无需安装** - 只需拖动到书签栏即可使用
- 🌐 **任何网站** - 在任何网页上都能使用
- ✏️ **即时编辑** - 点击元素即可编辑,实时预览效果
- 🎨 **样式调整** - 修改颜色、字体、大小、对齐等
- 🗑️ **删除隐藏** - 删除或隐藏不想要的元素
- 🖼️ **图片编辑** - 替换图片、上传本地图片、调整尺寸
- 🌐 **iframe支持** - 支持同源iframe内容编辑
- 📱 **跨浏览器** - 支持Chrome、Edge、Firefox、Safari等

## 📦 文件说明

- `bookmarklet.html` - 安装页面(推荐使用)
- `bookmarklet-source.js` - 源代码(可读版本)

## 🚀 快速开始

### 方法一:使用安装页面(推荐)

1. 用浏览器打开 `bookmarklet.html`
2. 将页面中的"🎨 网页随心改"按钮拖到浏览器书签栏
3. 打开任意网页,点击书签栏中的"网页随心改"
4. 开始编辑!

### 方法二:手动创建书签

1. 在浏览器中新建一个书签
2. 名称设为:"网页随心改"
3. URL设为:`bookmarklet.html`中的完整JavaScript代码
4. 保存后即可使用

## 📖 使用说明

### 启动编辑器

1. 打开任意网页
2. 点击书签栏中的"网页随心改"
3. 看到浮动工具栏表示启动成功

### 编辑元素

1. 点击页面上的任意文字或元素
2. 右侧会弹出编辑面板
3. 在面板中修改属性:
   - 文本内容
   - 字体大小
   - 文字颜色
   - 背景颜色
   - 字体粗细
   - 文字对齐
4. 修改实时生效

### 工具栏功能

- **✏️ 编辑模式** - 切换编辑模式开关
- **📸 截图** - 查看截图方法提示
- **✖️ 关闭** - 关闭编辑器(修改会保留)

### 编辑图片

1. 点击页面上的图片元素
2. 编辑面板会显示图片专属选项:
   - 替换图片URL
   - 上传本地图片(自动转Base64)
   - 调整图片宽度和高度
3. 也可以删除或隐藏图片

### 其他操作

- **删除元素** - 在编辑面板底部点击"删除元素"
- **隐藏元素** - 在编辑面板底部点击"隐藏元素"
- **关闭面板** - 点击面板右上角的 ✕ 按钮

## ⚠️ 注意事项

1. **临时修改** - 所有修改只在当前页面有效,刷新后会恢复原样
2. **保存方法** - 如需保存修改效果,请使用截图工具
3. **安全限制** - 某些网站(如Chrome商店、银行网站)有安全策略限制,可能无法使用
4. **本地显示** - 此工具仅修改本地显示,不会影响网站服务器
5. **合理使用** - 请不要用于欺诈或误导他人

## 🔧 技术细节

### 实现原理

1. 通过JavaScript注入样式和DOM元素
2. 使用事件委托监听页面点击
3. 动态修改元素的style属性
4. 使用innerHTML保存历史状态

## ❓ 常见问题

### Q: 为什么拖不到书签栏?
A: 确保书签栏已显示(`Ctrl+Shift+B`或`Cmd+Shift+B`),用鼠标左键按住拖动,不要松开直到拖到书签栏。

### Q: 点击书签后没反应?
A: 某些网站有安全限制。尝试在其他网站使用,或查看浏览器控制台是否有错误信息。

### Q: 如何保存修改?
A: 使用截图工具保存。修改只在当前会话有效,刷新页面会恢复。

### Q: 可以在手机上用吗?
A: 手机浏览器通常不支持书签小工具,建议在电脑上使用。

### Q: 修改会影响原网站吗?
A: 不会。所有修改只在你的浏览器本地显示,不会影响服务器或其他用户。

### Q: 支持哪些元素的编辑?
A: 支持文本元素(标题、段落等)和图片元素的编辑,可以修改样式、内容、尺寸等。
